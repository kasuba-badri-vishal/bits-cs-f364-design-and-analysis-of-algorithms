Graph1::Graph1 ( set< int >  V1)
inline

This is a Constructor for the Graph1 class.

This constructor is called by passing a list of vertices as argument. It creates a subGraph1 of the Graph1 specified by adj (adjacency list) using a subset of vertices.

### Example

Graph1(set<int> V1){
V = V1;
n_edges = 0;
for(auto x: V){
for(auto y: adj[x]){
if(V.find(y)!=V.end()) n_edges++;
}
}
}
