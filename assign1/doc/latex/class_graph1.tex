\doxysubsection{Detailed Description}
This program is the Implementation of {\bfseries{Divide\+\_\+and\+\_\+\+Conquer Algorithm}} presented in the research paper. 

We use D\+C\+SC, predecesser,descendent methods to find the {\bfseries{Strongly Connected Components}}.
